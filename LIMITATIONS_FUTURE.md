# Limitations and Future Applications - CVera 🚀

This document outlines the current boundaries of the CVera platform and our vision for its evolution.

## 1. Current Limitations
While CVera is a powerful tool for resume building, several constraints currently exist:
- **Template Variety**: Currently, the platform offers a curated but limited set of editorial templates. We aim to expand this into niche-specific layouts (e.g., Creative, Academic, Executive).
- **Language Support**: AI content generation and the UI are currently optimized for English.
- **Dynamic Content Styling**: While layout color can be changed, the font pairings and spacing are currently standardized to maintain brand integrity.
- **External API Dependency**: The system relies heavily on the availability of the Google Gemini API for its intelligent features.
- **Offline Mode**: A stable internet connection is required for AI generation and secure authentication.

## 2. Future Applications & Roadmap

### 2.1 Short-Term Improvements
- **Multi-template Selection**: Allow users to toggle between different layout philosophies (Modern vs. Classical) on the fly.
- **Enhanced Rich Text**: Expand the experience editor to support bullets, bolding, and hyperlinking more intuitively.
- **Profile Image Integration**: Better handling and cropping tools for professional profile pictures.

### 2.2 Long-Term Vision
- **Direct Job Board Sync**: Integrate with APIs like LinkedIn or Indeed to allow "One-Click Apply" using a CVera resume.
- **Multi-Language AI**: Expand content generation to support Spanish, French, German, and other major languages.
- **Real-time Collaboration**: Allow multiple users (e.g., a student and a career coach) to edit a resume simultaneously.
- **Mobile Companion App**: A dedicated mobile experience optimized for quick edits and recruiter interactions on the go.
- **Cover Letter Generator**: An AI-powered tool to generate tailored cover letters based on the saved CVera resume data.

## 3. Conclusion
CVera is not just a resume builder; it is a growing ecosystem of career tools. We are committed to continuous iteration based on user feedback and advances in AI technology to ensure that our users always have the strongest possible professional identity.
